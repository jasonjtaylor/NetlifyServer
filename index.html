37Sigs TAK Server

This is the landing page for the 37Sigs TAK (Team Awareness Kit) server.

Connecting TAK clients
  • Server name: 37Sigs
  • IP or domain: Use the address you entered in the TAK.gov configurator (local IP, DDNS hostname, or tunnel URL).
  • If you have a preference file from the configurator, import it in your TAK client to connect.

For access from outside the local network, the server is exposed via a public URL; see the repo for setup (tunnel or DDNS).
